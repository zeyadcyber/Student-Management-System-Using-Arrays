# Student Record Management System (C++)

This project is a Student Record Management System developed in C++ using dynamic arrays implemented from scratch without STL containers.

## Features

- Load student records from CSV files
- Add new student records
- Delete student records by ID
- Search students by ID or Name
- Display all student records
- Sort students by CGPA using Merge Sort
- Measure execution time using C++ Chrono Library
- Estimate memory usage
- Handle duplicate IDs and invalid inputs

## Technologies Used

- C++
- Dynamic Arrays
- Merge Sort
- Linear Search
- File Handling (CSV)
- Chrono Library

## Dataset Sizes Tested

- 500 records
- 2,000 records
- 8,000 records
- 30,000 records

## Restrictions

This project was implemented without using STL containers such as:

- vector
- list
- array
- algorithm
